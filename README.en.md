[中文](./README.md) | English

## Feature

- [x] register and log in
- [x] chat with others
- [x] view history
- [x] multiple chatroom
- [x] chat with robot
- [x] send pictures
- [x] send link
- [x] send emjio
- [x] preview picture
- [x] Message not read
- [x] Broken line reconnection

## Start

Dev Env: MongoDB、Node 8.5.0+、Npm 5.3.0+

Prod Env: Redis、MongoDB、Node 8.5.0+、Npm 5.3.0+

```Shell
npm install

npm run dev

```

## build

```
npm run build

npm run server
```
